# [fry-meme-generator](https://www.npmjs.com/package/fry-meme-generator)

### Install

npm install fry-meme-generator 

### Usage

```js
const frymeme = require('fry-meme-generator');

var meme = frymeme.fryme();

console.log(meme);

```
